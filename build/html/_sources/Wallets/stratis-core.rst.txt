**********************************************
Stratis Core
**********************************************

The all-in-one Stratis wallet allowing you to send and receive coins, secure the blockchain and get rewarded by using the Proof-of-Stake (PoS) network.  

Features
===========

* Inclusion of Staking capability using Proof-of-Stake (PoS)  
* Download of the entire Stratis Full Node blockchain  
* Supports the Stratis network with Block Validation and Transaction Relay  
* Aids Peer Propagation  
* Cold-staking will be enabled in future versions 

Customer Value
===============

* Secure the Stratis blockchain by running a full node  
* Stake 25k Strat and get rewarded for it  
* Wallet features to perform everyday transactions  

Target Audience
=================

Crypto-traders, medium/long-term investors, Strat stakers (aka HODL-ers)
